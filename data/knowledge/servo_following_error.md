# 伺服追隨誤差與位置誤差（Following / Position Error）

> 本文件為離線維修知識庫種子內容，供 RAG 檢索與 LLM 維護助理引用。內容為一般性
> 參考，實際處置須由現場合格人員判斷。

## 什麼是追隨誤差
追隨誤差（following error / position error）是「目標位置（rod_demand_pos）」與
「實際位置（rod_actual_pos）」之間的差。伺服系統運作時會有一定的動態誤差，但
**誤差持續變大**通常代表機構或控制出現問題。

## 常見原因
- 滾珠螺桿潤滑不足，導致摩擦上升、機構阻力增加。
- 滾珠螺桿或導螺帽磨耗、預壓下降，造成背隙與卡滯。
- 負載超出設計值或負載突變，使馬達無法即時跟上命令。
- 伺服增益（gain）設定不當或老化，控制響應變慢。
- 聯軸器鬆動、皮帶打滑，造成命令與實際運動脫節。

## 建議檢查項目
1. 檢查潤滑系統與油路，確認滾珠螺桿潤滑充足。
2. 手動盤車感受是否有局部卡滯或異音。
3. 檢視位置誤差隨時間的趨勢，確認是漸變（磨耗 / 潤滑）或突變（卡滯 / 負載）。
4. 確認負載與夾持力是否在規格內。
5. 必要時重新調整伺服增益，或請原廠協助。

## 與訊號特徵的對應
- `position_error_mean` / `position_error_max` 升高：追隨能力下降的直接指標。
- `torque_std` 升高：阻力不均、卡滯徵兆。
- `quadrature_rms`（Q 軸電流）升高：扭矩需求上升，常伴隨摩擦 / 負載增加。
