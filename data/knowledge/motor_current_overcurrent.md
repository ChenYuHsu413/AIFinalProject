# 馬達電流異常與過電流（Motor Current / Overcurrent）

> 離線維修知識庫種子內容。一般性參考，實際處置須由現場合格人員判斷。

## 三相電流異常（current abnormal）
正常運轉時三相電流（i_3p_a / i_3p_b / i_3p_c）應大致平衡，且隨負載合理變化。

### 常見異常與可能原因
- 三相不平衡：繞組匝間短路、接線鬆動、驅動器功率元件異常。
- 電流整體偏高（`current_rms` 上升）：負載增加、摩擦上升、機構卡滯。
- 電流尖峰：瞬間卡滯、負載突變、加減速過於急遽。

## 過電流（overcurrent）
過電流保護動作通常代表瞬時或持續電流超過驅動器上限。

### 可能原因
- 機構卡死或嚴重卡滯。
- 負載遠超設計值。
- 加減速時間設定過短，造成衝擊電流。
- 驅動器參數（電流上限 / 增益）設定不當。
- 馬達或電纜絕緣劣化、短路。

### 建議檢查項目
1. 確認機構可自由運動、無卡死。
2. 檢查負載與加減速設定是否合理。
3. 量測三相電阻與絕緣，排除繞組 / 電纜問題。
4. 檢視驅動器告警紀錄與電流上限設定。

## 與 D/Q 軸電流的關係
- Q 軸電流（`quadrature`）與扭矩輸出高度相關；負載 / 摩擦上升時 Q 軸電流上升。
- D 軸電流（`direct`）與磁場（勵磁）控制相關；D 軸異常多與弱磁或磁場控制策略有關。
